# Contributing

Thank you for your interest in contributing to Emblems Archive.

How to contribute
- Fork the repository and create a feature branch.
- Open a clear Pull Request describing your change and the reason.
- Include small, focused commits and reference any related issues.

Code style
- Keep HTML/CSS/JS simple and dependency-free.
- Use semantic HTML and accessible markup for content.

Review process
- Maintainers will review and request changes when necessary.
- All substantive changes should include a short explanation in the PR description.
